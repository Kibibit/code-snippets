How to use

import the following files to your html file:
<!-- in the head section --> <link rel='stylesheet prefetch' href='https://cdnjs.cloudflare.com/ajax/libs/highlight.js/9.12.0/styles/hopscotch.min.css'> <link rel="stylesheet" href="//cdn.rawgit.com/Kibibit/code-snippets/d6fa009d/kb-code-snippets.css"> <!-- at the end of the body section --> <script src='//cdnjs.cloudflare.com/ajax/libs/jquery/3.2.1/jquery.min.js'></script> <script src='//cdnjs.cloudflare.com/ajax/libs/highlight.js/9.12.0/highlight.min.js'></script> <script src='//cdnjs.cloudflare.com/ajax/libs/highlightjs-line-numbers.js/2.3.0/highlightjs-line-numbers.min.js'></script> <script src="//cdn.rawgit.com/Kibibit/code-snippets/d6fa009d/kb-code-snippets.js"></script>
then, simply inject a div with the following attributes:
<div class="kb-code <optional: terminal>" data-title="<snippet_title>" data-lang="<code_language>"> <!-- insert code here --> </div>
